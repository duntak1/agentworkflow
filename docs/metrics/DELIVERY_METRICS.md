# Delivery Metrics（DORA / Flow）

用途：记录交付效率与质量趋势，避免只看“任务完成”而看不到交付是否变快、变稳。

## 指标说明

| 指标 | 口径 |
|------|------|
| Deployment Frequency | 一段时间内成功发布到目标环境的次数 |
| Lead Time for Changes | 从需求 / commit / PR 到发布的时间 |
| Change Failure Rate | 发布后导致事故、回滚、hotfix 的比例 |
| MTTR | 从事故发现到恢复的时间 |
