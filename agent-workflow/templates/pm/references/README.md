# 参考资料上传区

请把资料放到以下目录。核心目录使用英文，文件名可以中文。

| 资料类型 | 目录 | 说明 |
|----------|------|------|
| PRD | `prd/` | 产品需求文档、功能说明 |
| UI 规范 | `ui/` | UI 规范、页面截图、设计说明 |
| 技术资料 | `tech/` | 技术方案、架构说明、第三方服务 |
| 接口资料 | `api/` | Swagger、OpenAPI、Postman、旧接口文档 |
| 业务规则 | `business/` | 权限、流程、枚举、状态机 |
| 附件 | `assets/` | 图片、表格、压缩包等 |
| 设计稿 | `design/` | Pencil 源文件、导出图、设计变更记录 |

资料上传后运行：

```bash
./scripts/aw pm intake-check --write
```
