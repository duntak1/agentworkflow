# 安全策略（仓库根摘要）

GitHub「Security」标签页读取**本文件**。完整条款与维护源：**[`agent-workflow/SECURITY.md`](./agent-workflow/SECURITY.md)**（请与该文件同步修改）。

## 报告漏洞

请勿在公开 Issue 中披露可利用细节（PoC、账号、内网地址等）。

请通过团队指定渠道私下报告：**（在此填写：安全邮箱、内部工单或责任人）**

## 范围与期望

- 本仓库可能包含协作文档与应用代码；报告时请说明受影响版本、commit 或 tag。
- 我们会在能力范围内确认、修复并协调发布说明（通常反映在 `CHANGELOG` 与安全相关提交中）。

## 密钥与敏感数据

禁止向仓库提交密钥、口令、生产连接串。若误提交，须立即轮换凭证并按团队事故流程处理。
